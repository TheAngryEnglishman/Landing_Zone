# Static homepage generator

## Installation

```bash
$ npm install
```

## Development

Run in dev mode - in two separate tabs:

```bash
$ npm run dev:client
$ npm run dev:site
```

## Build

Build for production - in two separate tabs:

```bash
$ npm run build:client
$ npm run build:site
```
